# Cable Price Intelligence

## Project Overview

This project aims to analyze the pricing of cable and wire products from different manufacturers in order to build a data-driven framework for:

- forecasting future product prices;
- comparing price levels across manufacturers;
- identifying comparable cable products based on their technical characteristics.

The project implements a complete data analytics pipeline, including automated invoice data extraction, data transformation, product normalization, exploratory data analysis, and machine learning-based price forecasting.

## Objectives

- Collect invoice data from multiple manufacturers.
- Extract product, pricing, and invoice information using Power Query.
- Normalize cable product descriptions into a unified structure.
- Compare equivalent products from different manufacturers.
- Analyze historical price dynamics.
- Develop predictive models for cable price forecasting.
- Visualize price trends and manufacturer comparisons.
